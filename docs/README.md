## Local Development

To run the documentation locally:

```bash
# Install dependencies
npm install

# Start dev server
npm run docs:dev
```

Visit `http://localhost:5173` to view the documentation.

## Building

Build the static documentation:

```bash
npm run docs:build
```

The built files will be in `docs/.vitepress/dist/`.

## Technology

This documentation is built with [VitePress](https://vitepress.dev/).
